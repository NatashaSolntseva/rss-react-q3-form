# React Forms (Portals • Uncontrolled • RHF • Zod)

A small demo app for the “React Forms” task:

- One reusable modal rendered via React Portal
- Two forms collecting the same data:
  - Uncontrolled (validate on submit)
  - React Hook Form (live validation via Zod)
- State stored with Redux Toolkit
- Accessible modals (focus trap, ESC to close, click outside)

## Tech Stack

- Vite • React • TypeScript
- React Hook Form, Zod, @hookform/resolvers
- Redux Toolkit, React-Redux
- Vitest, React Testing Library, JSDOM
